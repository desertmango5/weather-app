# Weather API App
## Displays local weather conditions

### Description
* Open Weather API
* Displays:
  * Tempurature
  * Conditions icon
  * Weather description
  * User's location
  * Air pressure
  * Humidity
  * Wind speed
* Toggles between imperial and metric
* Refresh button
* Background image changes to match weather conditions

### Technology Used
* HTML5
* Flexbox
* SASS
* JavaScript
* jQuery
